Climate Trace Project

The **Climate Trace** project focuses on analyzing environmental data, specifically emissions data, from different sources across the world. We leveraged the data available through APIs to trace emissions profiles and understand the global impact on climate change.

Steps in the Climate Trace Project:

  • Data Collection: We used the Climate Trace API to gather emissions data from a wide range of industries, countries, and sectors.
  • Data Analysis: Various techniques were applied to analyze the emissions data. This included breaking down emissions by sector and subsector, visualizing emissions trends, and identifying top polluting sources globally.
  • Geospatial Analysis: Using geospatial data, we mapped out the emission sources across different regions, giving us insights into local, regional, and global emissions patterns.
  • Reporting: We built interactive charts and reports that showcased emissions by sector, gases emitted, and the overall impact of these emissions.

Technologies and Tools Used:

  • API Integration: Climate Trace API was the primary source of emission data.
  • Python Libraries: Pandas, NumPy, and Matplotlib were used to process and visualize data.
  • Geospatial Tools: GeoPandas and Cartopy were used for visualizing the emission sources across geographical locations.

Outcome:

  • Created a detailed emission profile for various industries.
  • Mapped global emissions trends and identified major contributors to climate change.
  • Visualized emissions data with easy-to-understand plots and maps.

Google Earth Engine Project

The **Google Earth Engine** project focused on analyzing environmental data, such as temperature, air quality, and land cover, with a strong emphasis on satellite imagery analysis. Google Earth Engine (GEE) provides powerful cloud-based tools to analyze geospatial data and generate meaningful insights about environmental changes over time.

Steps in the Google Earth Engine Project:

  • Setup and Authentication: The first step was to set up a Google Cloud account and authenticate using the Earth Engine API. This allowed us to access satellite images and environmental data.
  • Data Processing: We processed data from different sources, such as MODIS (for vegetation indices) and other datasets available through GEE to assess environmental changes like deforestation, urbanization, and land degradation.
  • Analysis of Air Quality and Temperature: Using satellite-based data, we analyzed air quality patterns and temperature changes across different regions, with an emphasis on detecting trends in environmental degradation.
  • Visualization: After processing the data, we visualized the results on interactive maps, providing insights into long-term environmental trends.

Technologies and Tools Used:

  • Google Earth Engine API: The primary tool for analyzing satellite imagery and geospatial data.
  • Python: Used to interact with the GEE API, process data, and generate visualizations.
  • Jupyter Notebooks: Facilitated the step-by-step analysis, including visualizations of temperature, NDVI (Normalized Difference Vegetation Index), and land cover changes.

Outcome:

  • Detailed analysis of land cover changes, temperature trends, and air quality across various regions.
  • Identification of areas most affected by deforestation and urbanization.
  • Generated environmental maps that showcase the spatial distribution of key variables such as temperature and vegetation health.

Setup Details

For both projects, the following setup steps were essential:

Climate Trace Setup

  • Created an account on the Climate Trace platform and obtained API keys.
  • Installed the necessary Python libraries, including requests and Pandas, to interact with the Climate Trace API and process data.
  • Developed Python scripts to fetch emission data based on geographic regions and sectors.

Google Earth Engine Setup

  • Created a Google Cloud account and enabled the Earth Engine API.
  • Authenticated the API and installed the `earthengine-api` library in Python.
  • Used the GEE code editor and Python API to write scripts for accessing satellite data and performing analysis.

Outcomes and Results

The work completed under this project provided crucial insights into global emissions, land cover changes, and other key environmental metrics. Our findings will help in understanding the role of human activity in environmental degradation and provide a data-driven basis for policy decisions.

Climate Trace Outcomes:

  • Developed emission profiles for different sectors and regions.
  • Provided visual insights into emission trends over time.

Google Earth Engine Outcomes:

  • Generated maps and graphs for air quality and temperature trends.
  • Highlighted key areas of environmental change due to deforestation and urbanization.
