About this project

A browser-based Rock Paper Scissors game built to demonstrate interactive UI design and client-side game logic. Scores are tracked live across rounds, and both choices are displayed after each move.

  • Pick Rock, Paper, or Scissors — the computer responds instantly
  • Live scoreboard updates after every round
  • Visual choice display shows both sides of each match
  • Reset at any time to start a fresh session
