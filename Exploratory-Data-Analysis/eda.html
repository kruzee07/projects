Climate Trace Project

The Climate Trace project focuses on analyzing global emissions data. Using APIs and geospatial tools, we identified pollution sources and emission trends globally.

Steps in the Project:

  • Data Collection: Used Climate Trace API to gather emissions data across countries and industries.
  • Data Analysis: Applied breakdowns by sectors, visualized emission patterns, and tracked high-pollution regions.
  • Geospatial Mapping: Mapped emissions across the globe using GeoPandas and Cartopy.
  • Reporting: Built visuals of gases, sectors, and contributors.

Technologies:

  • Python (Pandas, NumPy, Matplotlib)
  • GeoPandas, Cartopy
  • Climate Trace API

Outcome:

  • Generated detailed emissions profiles
  • Identified leading global polluters
  • Built interactive plots and maps

Google Earth Engine Project

This project analyzed environmental change over time using satellite data and the Google Earth Engine (GEE). The focus was on temperature, air quality, deforestation, and urbanization trends.

Steps Taken:

  • Set up Earth Engine API and authenticated access
  • Analyzed data from MODIS, NDVI, and other sources
  • Mapped changes in land cover, temperature, and pollution

Tools Used:

  • Google Earth Engine API
  • Python + Jupyter
  • Satellite imagery analysis

Results:

  • Visualized temperature and vegetation shifts
  • Mapped deforestation and urban expansion
  • Identified areas with environmental risks

Setup Details

Climate Trace:

  • Registered and obtained API keys
  • Installed requests, pandas, and visualization libraries
  • Developed scripts to extract and format emissions data

Google Earth Engine:

  • Enabled GEE API via Google Cloud
  • Authenticated user token and installed Python SDK
  • Analyzed and visualized satellite data via code editor

Outcomes & Results

These projects provided data-driven insights on emissions and land-use change, helping visualize the global environmental impact.

Climate Trace:

  • Tracked sector-wise emissions globally
  • Built visual dashboards for trend analysis

Google Earth Engine:

  • Mapped temperature and air quality
  • Revealed regional impacts of urbanization and deforestation
