US Medical Insurance Costs

A Data Analysis Project

From: Codecademy

By: Krutarth Ghuge

Project Date: June 2024

Project Description

For this project, I explored a dataset on US medical insurance charges using Python. The dataset (insurance.csv) was analyzed using custom Python functions and visualized through graphs. The project was coded in Visual Studio Code and pushed to my GitHub repository.

Project Goals

  1. Evaluate how different attributes affect medical charges
  2. Determine cost differences based on health-related factors
  3. Explore correlations between children and charges
  4. Assess the impact of sex and region on charges

Data Overview

Columns:

  1. age
  2. sex
  3. bmi
  4. children
  5. smoker
  6. region
  7. charges

Summary: 1338 entries, no missing data, mix of numerical and categorical features.

Analysis Performed

  1. Smoker vs Non-smoker charge difference
  2. Charges vs Age
  3. BMI vs Charges by region
  4. Number of children vs charges

Evaluation

Each analysis included detailed calculations and visual plots. Key findings showed smokers have significantly higher average charges, charges increase with age, obesity correlates with higher charges, and a non-linear relation between children and charges.
